# Philosophy

Agent Context Economy is based on one idea: an AI coding agent should read a repository like a careful developer, not like a log scraper.

The agent should:

1. summarize before reading deeply,
2. pick likely files before opening them,
3. read the smallest meaningful window,
4. prefer symbols over scrolling,
5. expand only when the current context is insufficient,
6. avoid raw full output unless explicitly needed.

The goal is not to hide necessary context. The goal is to avoid irrelevant context.
